docker pull默认从Docker Hub（https://hub.docker.com/）下载镜像。也可以手动指定一个仓库地址拉去镜像。

如何上传镜像到dockerhub

首先在 Docker Hub 上注册一个账号：

  • 官网地址：https://hub.docker.com/

本地登录docker hub 帐号

为存在于本地的镜像打tag

docker tag hangge_server yuhang0385/hangge_server:v1

使用 docker push 命令将镜像上传到 Docker Hub：
docker push yuhang0385/hangge_server:v1

因为网速的问题，可以将得到的镜像上传到阿里云：参考链接：https://help.aliyun.com/document_detail/51810.html

拉取镜像：

docker pull yuhang0385/hangge_server:v1
